# Observatorio Cultural Independiente

Sitio institucional del Observatorio Cultural Independiente, una iniciativa de investigación, documentación y archivo con sede en Baños de Agua Santa, Ecuador.

## Propósito

Documentar procesos culturales contemporáneos, memoria territorial, turismo, patrimonio emergente, simbolización y gobernanza cultural mediante fuentes trazables y publicaciones abiertas.

## Sitio

Dominio previsto: `https://observatorioculturalindependiente.org`
